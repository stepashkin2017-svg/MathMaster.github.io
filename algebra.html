Алгебра

От уравнений до функций. Идеально для старта в математике.

1. Линейные уравнения

Линейное уравнение – это уравнение вида \(ax + b = 0\), где \(a \neq 0\).

Решение: \(x = -\frac{b}{a}\).

2. Квадратные уравнения

Квадратное уравнение: \(ax^2 + bx + c = 0\).

Дискриминант: \(D = b^2 - 4ac\).

Корни: \(x_{1,2} = \frac{-b \pm \sqrt{D}}{2a}\).

3. Функции и графики

Линейная функция: \(y = kx + b\). График – прямая.

Квадратичная функция: \(y = ax^2 + bx + c\). График – парабола.

Решите уравнение \(3x - 7 = 2\).
Найдите корни уравнения \(x^2 - 5x + 6 = 0\).
Вычислите значение функции \(y = 2x + 3\) при \(x = -1\).
